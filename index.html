ToDo List App

Despite being the most frustrating project I've worked on, the to-do list app I'm building for iOS is teaching me a lot about how to store data both on devices and in various databases as well as how to debug with Swift in XCode. My goals for this project are to be able to store to-do list data in a database and add a feature that allows users to create a schedule in which they can plan when to do certain tasks based on events that occur during their day (classes, work, etc.).

I wanted to build this app to make time management easier for myself. I like to write everything I have to do down however, since schedules often change I have to rewrite my daily/weekly plan several times. With this app I'll be able to just drag and drop to reorder my schedule.
